📊 Data Quality Inspector

Ferramenta de Análise Automatizada de Qualidade de Dados

🧠 Sobre o Projeto

Sistema desenvolvido em Python para análise automática de qualidade de dados em arquivos CSV, simulando um cenário real de Engenharia de Dados.

🎯 Problema Resolvido

Identificação de dados inconsistentes, valores ausentes e registros duplicados antes da análise ou uso em modelos.

⚙️ Funcionalidades

✔ Leitura de CSV
✔ Detecção de valores nulos
✔ Duplicados
✔ Score de qualidade
✔ Visualização de dados

🧠 Score de Qualidade

🟡 Bom (80.50)

O dataset possui qualidade aceitável, mas ainda requer tratamento de dados ausentes.

📈 Insight Gerado pelo Sistema

O dataset analisado apresenta baixa taxa de completude em colunas críticas, indicando necessidade de limpeza antes de uso em modelos preditivos.
